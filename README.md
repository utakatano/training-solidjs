# Training Solid

## Environment

- Node.js ... 16.3.0
- [Solid](https://www.solidjs.com/) ... 1.0.0

## Setup

If you want to create a app with typescript template,  
please run the following commands.

```sh
＃ npx degit solidjs/templates/ts {project_name}
＃ cd {project_name}
＃ npm install
＃ npm run dev
```
